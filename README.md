# Primer Avance
##Contexto
Para este primer avance de proyecto, decidí realizar una calculadora, debido a que me parece un proyecto complejo, en el cual podré demostrar las competencias que se necesitan para este primer semestre. Además considero que también podría reflejar todo lo aprendido en clases, en los exámenes y actividades realizadas a lo largo del semestre. En esta primera entrega realizare una parte del pseudocódigo, para posteriormente empezar a programarlo como un código.
